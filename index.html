Интервью: HTML

Что такое doctype?

Doctype - существует для указания типа документа. Добавляется первой стракой для любого HTML документа.

Что такое HTML?

Язык разметки с которым можно создавать веб страницы. Основная задача - это помочь браузеру отобразить определенную информацию.

Базовая структура HTML-документа

Doctype -> HTML - это корневая обертка страницы.

-> head - содержит все необходимые данные HTML документа.

-> body - содержит всю разметку HTML документа которая будет отображатся в браузере.

Базовая структура HTML-документа

Doctype -> HTML - это корневая обертка страницы.

-> head - содержит все необходимые данные HTML документа.

-> body - содержит всю разметку HTML документа которая будет отображатся в браузере.

Что такое валидация?

Валидация - это проверка на установленные веб стандарты и обноружение существующих ошибок. Эти стандарты называются спецификациями.

Основные типы проверок: проверка синтаксиса, проверка вложенности тегов, проверка на наличие посторонних элементов.

Для какого тэга используется атрибут alt?

Атрибут alt добавляется тэга картинки (img). Он нужен для того, чтобы в случае когда картинка не отображается на странице отображалось название картинки.

Типы input элементов в HTML

Input элемент существует для получения вводимых данных.

  • Текстовое поле (input type = "text")
  • Поле с паролем (input type = "password")
  • Поле с почтой (input type = "email")
  • Кнопка (input type = "button"/"submit") и тд

Интервью: CSS

Что такое CSS?

CSS - каскадная таблица стилей. Предназначены для добавления стилей для HTML документа.

Что такое CSS правило?

Селектор {Свойство: значение;}.

Свойство display - основные значения и как они работают?

Основные свойство:

  • None
  • Block
  • Inline
  • Inline-block
  • Flex

Inline - элементы расположены в одну строку и размер зависит от содержимого.

Block - размер определяется размерами заданных в стилях, но сам блок занимает всю свободную ширину экрана и все последующие блоки переносятся на следующую строку.

Inline-block - элементы расположены в одну строку, но можно задавать свой размер.

Flex - основная концепция в том, что мы можем задавать гибкие лейоуты, управлять направлением, одним свойством центрировать блоки по вертикали и по горизонтали, изменять порядок элементов в целом делать гибкие и отзывчивые макеты.

None - элемент ничего не показывает.

Что такое селектор и какие селекторы существуют?

Селектор - это часть CSS правила которая сообщает браузеру к какому элементу страницы будет приминен стиль.

Простые селекторы: class, id, универсальный селектор (*), селектор атрибута (a[href="test"]).

Составные селекторы: комбинация простых селекторов. Группавые селекторы (h1, h2, span), селектор потомка (div a), селектор псевдоклассов (hover) и тд.

Разница между Reset и Normalize

Практический все HTML элементы имеею стили по умолчанию (отступы, размер и тд). Каждый браузер приминяет различные стили.

Reset сбрасывает все стили по умолчанию, а Normalize стабилизирует все стили для различных браузеров.

Разница между class и id

class лучше использовать ради стилей, а для добавления логики лучше использовать id.

id - должен быть уникальным, class - можно задавать и использовать много раз.

у id вес больше.

Разница между адаптивном и отзывчивым дизайном?

В адаптивном - сайте есть несколько HTML, CSS, Js файлов одной страницы, эти файлы загружаются основываясь на устройство.

В отзывчивом - сайте есть только один набор файлов, но с помощью медиа (@media) запросов можно перестроить сайт на каждое разрешение экрана.

Что такое схлопывание границ?

Взаимодействие отступов по вертикали. Когда у двух элементов отступы разные и итоговое значение будет равно наибольшему.

Что такое кроссбраузерность?

Это когда сайт работает и выглядит одинаково в разных браузерах.

Keyframes для чего нужно и как использовать?

Keyframes - определяет ключевые кадры для анимации. Указывается начальная точка (from) и конечная (to). Ключевое свойство animation.

Свойства position

Основные свойство:

  • Static
  • Relative
  • Absolute
  • Fixed
  • Sticky

Static - значение по умолчанию.

Relative - сдвигает элемент относительно его обычного положения.

Absolute - пробадает из общего потока и для остальных элементов как будто он не существует.

Fixed - пробадает из общего потока и закреплен на экране.

Sticky - закреплен на экране, но в рамках заданного блока.

Что делает box-sizing: border-box?

При помощи box-sizing: border-box мы можем сказать браузеру, что ширина, которую мы ставим, относится к элементу полностью, включая border и padding.

Как обрабатывает WEB страницу браузер?

  1. Формируется DOM дерево из HTML
  2. Загружаются стили
  3. На основании DOM и CSS формируется дерево рендеринга
  4. Для каждого элемента рассчитывается положение на странице
  5. Отрисовка в браузере

Что такое вендорные префиксы?

Приставка к CSS свойству которая обеспечивает поддержку для браузера.

Разница между Progressive Enhancement и Graceful Degradation

Progressive Enhancement - это создание интерфейса от простого до сложного.

Graceful Degradation - это создание интерфейса от сложного до простого.

Что такое inline стили?

Inline стили - это стили которые пишутся напрямую в HTML. Имееют самый высокий приоритет.

Чем border отличается от outline?

border - является частью элемента.

outline - не является частью элемента. Просто визуальное состовляющее которое не влияет на блочную модель элемента.

Что такое БЭМ?

Блок Элемент Модификатор - разделяет интерфейс на независимые блоки. Позволяет быстро разрабатывать интерфейсы и повторно использовать существующий код.

В чем различие между строчными и блочными элементами?

Строчные - располагаются в одну строку.

Блочные - занимают всю свободную ширину.

В чем разница между padding и margin?

padding - внешние отспупы.

margin - внутренние отступы.

Что такое семантические элементы?

Они определяют конкретные части страницы. Позволяет браузеру ясно определить роль содержимого контента (header, nav, section, footer etc).

Свойство overflow

Используется чтобы убрать прокрутку на странице когда элемент выходит за границы страницы.

За что отвечают justify-content и align-items?

justify-content - влияет на расположение элементов по главной оси контейнера. По умолчанию по оси х.

align-items - влияет на расположение элементов по второстепенной оси, по оси y.

Приоритеты селекторов

Каждый селектор имеет свой вес и наибольший вес имееют inline-стили > id > class > тег.

Что такое псевдоклассы?

Псевдоклассы используются для определения характеристики элементов. Они не отображаются в исходном документе и не пренадлежат дереву документа DOM (hover, focus etc).

background-size

cover - сохраняет свои пропорции, но изменяется размер чтобы уместиться в блок.

contain - ужимает картинку пока не вместиться в блок.

Что такое псевдоэлементы?

Псевдоэлемнты - добавляются на селектор и позволяют стилизовать определенный элемент (before, after, first-letter etc).

!important для чего нужен?

Чтобы повысить приоритет селектора и переопределить стили.

Разница между script, script async и script defer

script - По стандарту браузер загружает первым, прогрузка происходит сверху вниз, синхронно.

async/defer - браузер прогружается в асинхронным режиме, defer используют когда важен порядок загрузки скриптов и срабатывает только когда заргузился весь HTML.

Для чего рекомендуется использовать атрибуты data?

Они предназначены для хранения каких либо значении. Например: для создание всплывающих подсказок без приминения js (data-название).

Что такое селектор атрибутов?

Устанавливает стиль для элементов, у которых присутствует указанный атрибут. В качестве селектора используются квадратные скобки (input[type: password]), внутри которых пишется произвольный атрибут.

Для чего используются препроцессоры?

Препроцессоры - расширяет стандартные возможности CSS. Например: миксины. Миксины - это определенные свойства которые позже можно будет переиспользовать.

Интервью: JavaScript

Какие есть типы данных в Js?

  • "str" — строка.
  • 123 — числа.
  • true/false — булеаны.
  • BigInt — для больших чисел.
  • null — представляет собой «ничего», «пусто» или «значение неизвестно.
  • undefined — Оно означает, что «значение не было присвоено.
  • object — В объектах хранят коллекции данных или более сложные структуры.
  • symbol — используется для создания уникальных идентификаторов в объектах.
  • typeof — Оператор typeof возвращает тип аргумента.

Методы строк в Js

  • str.length — чтобы узнать длину строки.
  • str.toUpperCase() — возвращает значение строки, преобразованное в верхний регистр.
  • str.toLowerCase() — возвращает значение строки, преобразованное в нижний регистр.
  • str.replace() — заменяет значение в строке.
  • str.indexOf(), str.lastIndexOf() — ищет указанный текст и возвращает числовой индекс если найден, а если нет -1.
  • str.slice(), substr(), substring() — для извлечения подстроки из строки.

Методы массивов в Js

  • arr.length — чтобы узнать длину массива.
  • arr.push() — добавляет элементы в конец.
  • arr.unshift() — добавляет элементы в начало.
  • arr.pop() — извлекает элемент с конца.
  • arr.shift() — извлекает элемент с начала.
  • arr.join() — превращает массив в строку.

Для чего используется директива «use strict»?

«use strict» — это директива, которая заставляет весь наш код или код отдельной функции выполняться в строгом режиме.

Он заменяет исключениями несколькие ошибки которые Js пропускает по умолчанию.

Преобразование типов

String() - Происходит, когда нам нужно что-то вывести.

Number() - Происходит в математических операциях.

Boolean() - Происходит в логических операциях.

В чем разница между операторами "==" и "==="?

"==" нестрогое равенство сравнивает значения, а "===" строгое равенство допольнительно сравнивают типы.

Какими способами можно объявить переменную?

  • Название переменной и значение (a = 5).
  • С помощью var (var a = 5). Имеет глобальную или функциональную область видимости, но не имеет блочную.
  • С помощью let (let a = 5). Имеет блочную область видимости.
  • С помощью const (const obj = {}). Имеет блочную область видимости.

Разница между function declaration и function expression?

Function declaration - это функция которая создется в потоке документа (function name (a,b){}).

Function expression - это когда созданная функция присваивается в переменную.

Основное отличие - это Function declaration создается до выполнения кода.

Что такое поднятие (Hoisting)?

Поднятие — это термин, описывающий подъем переменной или функции в глобальную или функциональную области видимости.

JavaScript сначала объявляет и затем инициализирует переменные. let, const не всплывают как var.

Операторы && и ||

Оператор && - находит и возвращает первое ложное значение или последнее если все значения истинны.

Оператор || - находит и возвращает первое истинное значение.

Типы таймеров

setTimeout() - позволяет вызывать переданную функцию один раз по истечение времени.

setInterval() - позволяет вызывать фунцию постоянно через определенное время.

В чем разница между null и undefined?

undefined неожиданное отсутствие значения. Переменная без значения выдает undefined.

null — умышленное отсутствие значения. Присваем самостоятельно.

Разница между var, let, const?

var - переменные могут быть как обновлены, так и переопределены внутри области видимости.

let - переменные можно обновлять, но не переопределять.

const - переменные нельзя ни обновлять, ни переопределять.

forEach, map, filter, reduce для чего нужны?

forEach - берет колбэк функцию и запускает на каждый элемент массива. Ничего не возвращает, используют для перебора элементов.

filter - используется для отфильтровки массива по определенному критерию. Разница между forEach и filter в том, что forEach пускает цикл по массиву и выполняет колбэк, filter выполняет колбэк и проверяет возвращаемое значение.

map - как и forEach, filter берет колбэк и пускает его по всем элементам массива, но то что делает его уникальным - создает новый массив, основываясь на существующим.

reduce - принимает функцию редюсер каждому элементу массива и в результате вернет одно значение.

Разница между Map и Set?

Map – это коллекция ключ/значение, как и Object. Но основное отличие в том, что Map позволяет использовать ключи любого типа.

Set – это особый вид коллекции: «множество» значений (без ключей), где каждое значение может появляться только один раз.

Разница между стрелочными функциями и обычных?

  • У стрелочных функции нету arguments.
  • Разный синтаксис.
  • У стрелочных функции нету this.
  • Нельзя вызвать с помощью new.

Операторы spread и rest?

Spread - оператор расширения, используется в вызове функции или где-либо ещё. Он извлекает элементы из массива.

Rest - остаточные параметры, располагается в конце списка аргументов функции. Он собирает остальные неуказанные аргументы и делает из них массив.

Что такое this?

Для доступа к информации внутри объекта можно использовать ключевое слово this.

Что такое псевдомассив arguments?

Это коллекция аргументов которые передаются в функцию. Это объект подобный массиву у которого есть только свойство length.

Разница между .call(), .apply(), .bind()?

Каждый в качестве первого аргумента принимает контекст вызыва.

Разница между .call() и .apply() в том как передаются аргументы внутри функции, .call() - перечисление через запяту, .apply() - через массив.

.call() и .apply() вызываются на месте в момент когда их принимают, а .bind() создает функцию обертку которую можно присвоить в переменную.

Разница между event.target и event.CurrentTarget?

event.target - это элемент в котором происходит событие.

event.CurrentTarget - это элемент которому прикреплен прослушиватель событии.

В чем разница между методами event.preventDefault() и event.stopPropagation()?

Метод event.preventDefault() отключает поведение элемента по умолчанию.

Метод event.stopPropagation() отключает распространение события (его всплытие или погружение).

В чем разница между методами event.stopImmediatePropogation() и event.stopPropagation()?

event.stopPropagation() - отменяет дальнейшее всплытие.

event.stopImmediatePropogation() - отменяет дальнейшее всплытие и на текущем элементе.

Что такое замыкание (Closures)?

Это функция, которая запоминает свои внешние переменные и может получить к ним доступ.

На хранение таких переменных выделяется определенная память и когда функция заканчивает выполнение, память очищается.

Что такое Деструктуризация?

Деструктуризация - это специальный синтаксис, который позволяет нам «распаковать» массивы или объекты в несколько переменных.

Что такое область видимости (Scope)?

Область видимости — это место, где (или откуда) мы имеем доступ к переменным или функциям. JS имеем три типа областей видимости: глобальная, функциональная и блочная (ES6).

Что такое IIFE?

IIFE или Immediately Invoked Function Expression — это функция, которая вызывается или выполняется сразу же после создания или объявления.

Разница между host и нативными объектами?

host объекты предоставляются браузером (window, document etc).

Нативные объекты это объекты JavaScript (string, Math etc).

Что такое NaN?

Not a number. Оно получается кода математическая функция сработало не верно (parseInt("str")).

Что такое функции высшего порядка?

Это функции которые возвращают другие функции или же принимает другую фунцию (map, reduce etc).

Что такое такое чистая функция?

Концепция функционального программирования, в ней не должно быть побочных эффектов (http, dom запросы) и каждый раз возвращает одинаковый результат.

В чем разница между синхронными и асинхронными функциями?

Синхронными - называют блокирующие функции, а асинхронные нет.

Что такое всплытие событий?

Событие «всплывает», переходит от элемента, вызвавшего событие (event.target), к его родителю, затем поднимается еще выше, к родителю родителя элемента, пока не достигает объекта Window.

Что такое погружение событий?

В фазе погружения событие опускается от объекта Window до цели события через всех его предков.

Что такое делегирование событий?

Это прием разработки когда можно добавить один обработчик на общего предка.

Что такое DOM?

DOM или Document Object Model — это объектная модель документа которую браузер создает в памяти компьютера на основании HTML документа.

Методы поиска элементов в DOM?

  • document.getElementById("id")
  • document.getElementByName("name")
  • document.getElementByTagName("div")
  • document.getElementByClassName("class")
  • document.querySelector("nav + p")
  • document.querySelectorAll("nav > ul > li")

Что такое распространение события?

Это механизм который срабатывает когда происходит какое-то событие в документе. От window до элемента.

Основные фазы распространения события:

  • Фаза захвата - от корня документа и проходит по DOM дереву до элемента.
  • Таргет - событие достигает целевого элемента.
  • Всплытие - событие возвращается обратно до window.
